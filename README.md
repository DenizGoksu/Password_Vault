# Password Vault
C# ve SQLite kullanılarak geliştirilmiş, XOR şifreleme destekli bir şifre yönetim panelidir.

#  Özellikler
- CRUD Operasyonları: Veri ekleme, okuma, güncelleme ve silme.
- SQLite Entegrasyonu: Verilerin yerel veritabanında kalıcı olarak saklanması.
- XOR Maskeleme: Şifrelerin güvenliği için temel seviye kriptografik işlem.

#  Kullanılan Teknolojiler
- .NET 8.0
- Microsoft.Data.Sqlite
- Newtonsoft.Json

#  Kurulum ve Çalıştırma
-Projeyi bilgisayarınıza indirin: git clone https://github.com/DenizGoksu/Password_Vault.git
-Proje klasörüne girin: cd Password_Vault
-Gerekli kütüphaneleri yükleyin: dotnet restore
-Uygulamayı çalıştırın: dotnet run
